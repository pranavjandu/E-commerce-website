Project overview:
Mobile E-commerce website using PHP and mySQL


Steps to view project:

1. Copy the folder webtek into C:/xampp/htdocs
2. Start apache server and mysql using xampp
3. go to localhost:8080 (default) and go to PHPmyAdmin.
4. Create a Database named 'testing' if you do not have it already.
5. Now go to url : localhost:8080/webtek/database/import.php
   This will create the tables in your database.
6. Go to url : localhost:8080/webtek
